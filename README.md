# 💼 PayrollPro – Employee Payroll System

## 📌 Project Description

PayrollPro is a simple web-based Employee Payroll System that helps manage employee salary details, calculate payroll, and display reports in a clean dashboard interface.

This project is developed as a **Mini Project in C (with frontend UI)** to demonstrate payroll processing concepts.

---

## 🚀 Live Demo

👉 https://suriya310.github.io/C-MINI-PROJECT-PAYROLL/

---

## ✨ Features

* ➕ Add Employee (ID, Name, Basic Pay, OT Hours)
* 💰 Calculate Gross Pay
  `Gross = Basic Pay + (OT Hours × OT Rate)`
* 📊 Tax Deduction (based on simple rules)
* 💵 Net Pay Calculation
* 📋 Dashboard View:

  * Total Employees
  * Total Payroll
  * Average Pay
  * Total Tax Collected
* 🧾 Employee List Display
* 🎨 Modern UI with responsive design
